#### 小程序数据绑定

- 基础数据绑定：

  ```
  <!--pages/bind/bind.wxml-->
  <text>name:{{studentName}}</text>
  ```

  ```
    data: {
        studentName:'tom'
    },
  ```

  `data`是页面的初始数据，数据绑定的所有接口都由`data`和外部进行对接。

    ```
    //html
    <view id="item{{id}}">
        item-view
    </view>
    
    
    //js: 
    data: {
          studentName:'tom',
          studentAge:'20',
          flag:true,
          id:2
      },
      
      
    //css
    #item1{
      color: yellow;
      font-size: 50rpx;
    }
    #item2{
      color: orange;
      font-size: 30rpx;
    }
    ```
